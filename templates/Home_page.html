List Route of Demo:

  1. /get_all_book

    Route:/get_all_book
    Method:GET
    Use: Get all book of Database

  2. /get_book_id

    Route:/get_book_id
    Method:GET
    Use: Get ID book of Database
    Request Header-ContentType:application/json
    Request body:
    {
    "ID":Value(Integer)
    }

  3. /update_book_id

    Route:/update_book_id
    Method:PUT
    Use: Update Quantity of ID book
    Request Header-ContentType:application/json
    Request body:
    {
    "ID":Value(Integer),
    "QUANTITY":Value(Integer)
    }

  4. /delete_book

    Route:/delete_book
    Method:DELETE
    Use: Delete ID book of Database
    Request Header-ContentType:application/json
    Request body:
    {
    "ID":Value(Integer)
    }

  5. /add_book

    Route:/add_book Method:POST
    Use: Add ID book of Database
    Request Header-ContentType:application/json
    Request body:
    {
    "ID": Value(Integer),
    "NAME": Value(String-49 char),
    "PRICE": Value(Float),
    "QUANTITY": Value(Integer)
    }
